title: "What is Makx?"
description: "A meme token launchpad with fair launches, automatic DEX migration, and on-chain identity."
---

Makx is a meme token launchpad built on Uniswap V4. Creators launch tokens with a bonding curve that automatically migrates liquidity to Uniswap once the token sells out — no manual steps, no rug pulls.

Every token gets a unique ENS subdomain (like `pepe.makx.eth`), making it discoverable on-chain without a backend.

## Three ways to launch

<CardGroup cols={3}>
  <Card title="Standard Launch" icon="rocket">
    A single creator deploys a token. They can buy first in the same transaction — sniper-proof. Once all tokens sell, liquidity moves to Uniswap and the LP is locked forever.
  </Card>
  <Card title="Prepool Launch" icon="users">
    A group of people pool ETH before the token exists. Everyone enters at the same price. The LP stays in the Prepool contract, earning trading fees for contributors — forever.
  </Card>
  <Card title="Agent Launch" icon="robot" href="/agent-launcher">
    Deploy an AI agent that monitors social feeds for trending memes and launches coins autonomously. Your agent detects, launches, and promotes — 24/7, no manual intervention.
  </Card>
</CardGroup>

## What makes it different

- **No rugs** — LP is either burned (standard) or locked in a contract (prepool). Nobody can pull liquidity.
- **Fair entry** — Dev buys happen atomically. Prepool launches give everyone the same price.
- **On-chain identity** — Token metadata lives in ENS records, not a database.
- **Automatic graduation** — When the bonding curve sells out, Uniswap liquidity is created in the same transaction.
